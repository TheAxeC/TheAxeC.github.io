    • Cardinal: scripting language
     Jan 2015 - Sept 2015

    Cardinal is a small, fast, class-based, Object Oriented scripting language written in C. It is built upon the skeleton of an existing scripting language and shows how I can modify and improve existing software, as well as design new components to this software. New components include: a debugger, an embedding API, multiple inheritance and a new module system.
